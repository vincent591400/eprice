# ePrice Query Tool

查詢 ePricebook 產品價格，並從 anydoor 取得 Cost(Highest)。

## 需求

- Python 3.8+
- Chrome 瀏覽器

## 安裝

```bash
pip install -r requirements.txt
```

## 使用方式

### 視窗版 (GUI)

```bash
python eprice_gui.py
```

### 網頁版

```bash
python web_app.py
```

瀏覽器開啟 http://localhost:5000

### 開機自動啟動網頁版

以系統管理員執行 `install_autostart.bat`
